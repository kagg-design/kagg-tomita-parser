Tomita Parser plugin supports WordPress search with Tomita Parser by Yandex. This sophisticated tool for linguistic analysis allows extract phrases from the text basing on linguistic text parser.

== Description ==

After activation of the plugin, standard WordPress search results will contain only those whole phrases which contain search string. Search string is highlighted.

Plugin requires php proc_open function to be enabled on server.

== Installation ==

= Minimum Requirements =

* PHP version 5.5 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* php proc_open function enabled

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Tomita Parser plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Tomita Parser” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
